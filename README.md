#### 回溯算法

解题步骤：

1.针对所给问题，定义问题的解空间，至少包括问题的**一个最优解**。

2.确定**易于搜索的解空间结构**，可以使用回溯法方便的搜索整个解空间结构

3.使用深度优先(DFS)搜索解空间，并在搜索的过程中使用**剪枝函数**避免无效的搜索回
